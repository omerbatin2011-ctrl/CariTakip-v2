# DAL ERP Widget Library v137

Bu klasör DAL ERP Next tasarım sisteminin ortak bileşenlerini içerir.

## Temel bileşenler

- `ERPPage`
- `ERPPanel`
- `ERPCard`
- `ERPButton`
- `ERPToolButton`
- `ERPToolbar`
- `ERPStatCard`
- `ERPResponsiveGrid`
- `ERPTable`
- `ERPSearchBox`
- `ERPLineEdit`
- `ERPComboBox`
- `ERPDateEdit`
- `ERPForm`
- `ERPDialog`
- `ERPStatusBar`

## Kural

Yeni ekranlarda doğrudan `QPushButton`, `QTableWidget`, `QLineEdit` üretmek yerine mümkün olduğunca bu bileşenler kullanılacak.
